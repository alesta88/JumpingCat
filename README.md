## ルール
***
ジャンプして高みを目指すお手軽ゲームです。   
かわいい猫がプラットホームからプラットホームへと跳び移ります。  
下に落ちたらと死んでしまう。危ないプラットフォームがあるので、ぶつからないようにジャンプしましょう!


## 操作方法
***
ジャンプは自動で行われる、矢印キーでそれぞれ左右に移動します。
